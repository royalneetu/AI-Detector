# AI vs Real Image Detector

## Project Description
A deep learning model that detects whether an image is Real or AI Generated using CNN and MobileNetV2.

## Tech Stack
- Python
- TensorFlow / Keras
- MobileNetV2 (Transfer Learning)
- Streamlit
- OpenCV

## Features
- Upload any image (JPG, JPEG, PNG)
- Detects Real vs AI Generated
- Shows confidence percentage

## Model Details
- Dataset: 1,00,000 images (50k Real + 50k Fake)
- Architecture: MobileNetV2 + Custom layers
- Accuracy: 90%+

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
```
